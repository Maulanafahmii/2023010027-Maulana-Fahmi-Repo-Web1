Visi & Misi Program Studi RPL

Visi: Menjadi program studi unggulan dalam pengembangan perangkat lunak berbasis teknologi terbaru.

Misi:

  • Menyelenggarakan pendidikan berkualitas di bidang rekayasa perangkat lunak.
  • Meningkatkan penelitian dan inovasi di bidang perangkat lunak.
  • Memperluas kerja sama dengan industri perangkat lunak.
